# 🚀 AWS EC2 ウェブサーバープロジェクト

## 📋 概要
AWS EC2のフリーティアを使用した、セキュアで本番環境対応のウェブサーバーの完全なデプロイ。

## 🌐 ライブデモ
**ウェブサイト:** http://54.85.190.169

## 🛠️ 構築内容
1. **AWSインフラストラクチャ:** EC2 t3.microインスタンス（フリーティア）
2. **ウェブサーバー:** Amazon Linux 2023上のApache HTTPサーバー
3. **セキュリティ:** SSHアクセス制限、適切なファイアウォールルール
4. **バックアップ:** 災害復旧用のAMIスナップショット作成
5. **監視:** 基本的なシステムヘルスチェックの実装
